# MonSiteCom.com — Plateforme de création de sites artisans

## Fichiers
- `index.html` — Page d'accueil marketing
- `editeur.html` — Éditeur visuel complet (type Wix)
- `CNAME` — Domaine monsitecom.com pour GitHub Pages

## Déploiement GitHub Pages
1. Créer repo public GitHub
2. Uploader tous les fichiers
3. Settings → Pages → Deploy from main
4. Custom domain → monsitecom.com

## Fonctionnalités éditeur
- Choix 3/6/9 pages au démarrage
- 13 types de blocs (hero, services, galerie, avis, tarifs, blog, équipe...)
- Upload de photos depuis l'ordinateur
- Modification textes en direct (clic)
- Couleur et police globales personnalisables
- Propriétés par section (bg, couleur texte, colonnes...)
- Prévisualisation desktop/tablet/mobile
- Score SEO en temps réel
- Mentions légales + RGPD auto-générées
- Export HTML par page (tous les fichiers)
